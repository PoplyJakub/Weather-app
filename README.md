## Settings

Create file **_.env_** in this poject (no in src directory)

Put your API key from OpenWeather to VITE_WEATHER_SECRET_API_KEY in **_.env_** file of this project.
Example

```
VITE_WEATHER_SECRET_API_KEY = //your API key
```

#### Run

```
npm install
```

```
npm run dev
```
